### Move

A Spring Boot ride-hailing backend service that handles ride matching, location tracking, and real-time event streaming via Kafka, Redis, PostgresSQL.

#### Features

- Request and manage rides
- Real-time driver location tracking
- Nearby driver discovery
- Ride status management
- Event-driven architecture using Kafka
- Audit logging

#### Tech Stack

- Java, Spring Boot, Redis, PostgreSQL, Kafka
